Streambake

A Perl-based system for sending audio files to an HTTP streaming server like Icecast and/or Shoutcast

View On GitHub
Downloads:
  • ZIP
  • TAR

    Using different sources (file, database, iPod, etc.), encode files into MP3/OGG/FLAC format for streaming to an Shoutcast/Icecast HTTP streaming server.

    You can get support with Streambake by posting to the Streambake mailing list streambake@googlegroups.com. You can browse old mailing list messages at http://groups.google.com/group/streambake.
